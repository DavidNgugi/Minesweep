Minesweep

Simple JavaScript minesweeper game

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Minesweep Github Page.

This is a simple JavaScript Minesweeper Game. The objective is to move around the board by clicking on each box without hitting 3 or more mines. The more non-mines you find, the more your score will be.

It's survival in that 5*5 area! No messing around or you die!
